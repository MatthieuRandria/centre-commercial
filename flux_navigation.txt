→ Changer statut → Notification client automatique

FLUX 5 - AJOUT PRODUIT (Manager):
Dashboard boutique → Mes produits → Nouveau produit → Formulaire complet
→ Upload images → Enregistrer → Publication

FLUX 6 - VALIDATION BOUTIQUE (Admin):
Login admin → Dashboard → Alerte inscription → Liste boutiques
→ Click "Valider" → Modal validation → Attribuer emplacement
→ Confirmer → Email envoyé au manager
